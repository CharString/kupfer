Disparadores

Complemento «Disparadores»

Con Disparadores puede coger un comando que ejecute habitualmente en Kupfer, como lanzar una aplicación o abrir un documento, y asociarle una combinación global de teclas para ejecutarlo. Mientras Kupfer esté en ejecución, cualquier aplicación puede tener el foco actual.

Uso básico

Crear un disparador nuevo

  1. Configure en Kupfer el comando que quiere ejecutar

  2. Pulse Ctrl+Intro para crear un comando compuesto.

  3. Pulse Tab y seleccione la acción Añadir disparador... y pulse Intro.

  4. Usando la ventana que aparece, pulse las teclas modificadores y un número o una letra para establecer un atajo globa. Un ejemplo es pulsar y mantener Ctrl+Alt y entonces pulsar y soltar R para asociar Ctrl+Alt+R.

Quitar un disparador

  1. Encuentre la fuente de Disparadores en Kupfer y pulse → para ver su contenido.

  2. Busque el disparador activo en la lista y use la acción Quitar disparador.

Ya que los disparadores de atajos de teclado son globales, se puede usar desde cualquier aplicación. Seleccione los atajos con cuidado para que no entren en conflicto con otras funciones.

Consejos para usuarios avanzados

Los Objetos del proxy como Texto seleccionado, Archivo seleccionado, Ventana de primer plano, Último comando etc. admiten disparadores muy potentes.

La acción de aplicación Lanzar es especial; si asocia un disparador usando Lanzar, iniciará la aplicación si no está en ejecución, pero dará el foco a la aplicación cuando esté en ejecución. Por el contrario, Iniciar de nuevo iniciará la aplicación tanto si está ejecución como si no.

Ejemplo de «Disparadores»

Editor de texto → Lanzar

Inicie el Editor de texto, o dé el foco a su ventana si ya está en ejecución.

Text seleccionado → Buscar con → Google

Buscar en la web usando el texto seleccionado actualmente.

Archivo seleccionado → Mover a → (Carpeta de descargas)

Mover el archivo seleccionado actualmente a una carpeta específica

Canciones → Buscar contenidos

Incluso puede añadir un disparador para abrir un subcatálogo específico en Kupfer, por ejemplo, canciones de Rhythmbox.
